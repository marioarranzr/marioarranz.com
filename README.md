## Running in development
`gatsby develop [-p 8000]` or `yarn develop`

## Trouble solving
- Remove node_modules folder `rm -R node_modules/`
- `npm install`
- Remove package-lock.json `rm package-lock.json`
- `yarn install` # Better and faster than npm
